Code of Conduct

We want WaffleJS to be a safe and inclusive environment. That’s why ask our attendees to follow our code of conduct.

In the unfortunate case that you see someone violating the code of conduct, here are some ways you can report it to us:

  • DM @wafflejs.
  • Let a co-organizer know, in person or via Twitter. ( Visnu, Billy, Emily, Kelly, Max, Katrina )
  • Anonymously text (650) 735-0005.
  • Or, anonymously report it via the form below.

We’ll review and act on it. If you let us know who you are, we’ll follow up with you.
